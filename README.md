## 💡 Why PRISM?

Finding the right professor for research collaboration is **slow, inefficient, and inaccurate**.

PRISM solves this using **AI + NLP + intelligent matching**, enabling users to:

✅ Discover experts using natural language
✅ Match projects with professors instantly
✅ Analyze research domains and expertise deeply


### Frontend

```bash
cd frontend
npm install
npm start
```
